# Minecraft-DirectPickUp-Plugin
[Paper/Bukkit] 1.21.x DirectPickUp Plugin by OZI09

Version of Plugin: 1.0
Version Minecraft: 1.21.x
Version Tested Server: 1.21.10 Bukkit

Tired of diamonds falling into lava while mining?
A plugin that allows dug-up items to be immediately placed in the inventory instead of falling on the ground.
